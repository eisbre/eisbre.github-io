##下载

官方通道下载慢，建议使用国内的镜像网站

安装

传送门

安装时跳过登录，选择对应需求的包，选中安装。

创建项目

点击创建项目后，选择项目路径以及 给项目起名称

名称 - 不能有中文 不能有空格

路径 - 不能有中文路径

默认创建有窗口类，myWidget，基类有三种选择： QWidget 、QMainWindow、QDialog

一般不确定类型，选最大的QWidget ，确定时再选择派生的QMainWindow、QDialog

类名可以自定义，界面创建可以不选，暂时只用代码实现界面不会用到

创建一个窗口

默认创建的文件已经包含基本创建窗口的操作，构建运行即可

我们先来看main函数

QApplication a 应用程序对象，有且仅有一个

myWidget w;实例化窗口对象

w.show()调用show函数 显示窗口

return a.exec() 让应用程序对象进入消息循环机制中，代码阻塞到当前行

创建按钮

我们需要先引入QPushButton

QPushButton * btn = new QPushButton;使用new创建一个按钮

btn->show();同样，按钮也需要show才能显示

但此时按钮与原窗口分离，我们需要把他们绑定在一起

btn->setParent(this);设置依赖
btn->setText(“按钮”);设置按钮显示的文字

这里是一些重置窗口，按钮大小，标题，位置的函数

QPushButton * btn = new QPushButton(“1”,this);相比之前的创建方式，此类创建方式代码更简洁，但会按照控件大小创建按钮，参量分别为显示信息，依赖对象

Qt中的坐标系左上角为 0 ， 0 点 ，x以右为正方向，y以下为正方向

效果

对象树（了解）

当创建的对象在堆区时候，如果指定的父亲是QObject派生下来的类或者QObject子类派生下来的类，可以不用管理释放的操作，将对象会放入到对象树中。

一定程度上简化了内存回收机制

信号和槽

connect函数

参数1 信号的发送者

参数2 发送的信号（函数地址）

参数3 信号的接受者

参数4 处理的槽函数 （函数的地址）

自定义信号和槽

我们要实现这样一个功能：

下课后，老师触发一个饿了信号，学生响应信号，请客吃饭

在这里可以创建新的文件，对应存放新的对象及其函数

首先要在头文件里声明对应的函数

teacher.h

作为信号存在，需要在signals下加入函数

自定义信号 写到signals下
返回void只需声明，不需要实现
可以有参数，可以重载

student.h

最为执行的动作，需要定义实现槽函数

早期版本必须写到public slots下，高版本可以写到public，也可以全局
返回void，需要声明，也需要实现
可以用参数，可以重载

widget.h

这个头文件相当于汇，总类似于主函数

我们需要在private中定义用到的类，以及触发函数（使连接函数得以执行的函数）

teacher.cpp

因为信号函数不需要实现所以并不需要我们在teacher.cpp添加函数，所以保持初始

student.cpp

因为槽函数需要实现，所以我们要在student.cpp中实现student.h中声明的槽函数。

main.cpp

只起到调用作用，不需要定义任何东西，保持初始

widget.cpp

主要内容在这里，其实这里只是在main中展示了一个窗口，而widget中主要是实现了创建类到调用，完成一次信号的触发到响应

注意：这里在代码的执行顺序上可能会有歧义，connect 函数的作用是连接，形成一个过程，并不是直接执行。我们想触发这个过程，得需要一个触发

，就和之前的鼠标点击一样。而在触发之前，我们需要先把这种连接创建出来，才能触发。所以先是connect，然后才是 classIsOver。

信号和槽拓展

当我们的程序运行起来后会直接调用下课，触发后面的响应。但这不是我们所希望的。现在我们要创建一个按钮，当我们点击这个按钮时，才会触发后面的响应。

我们只改变需widget.cpp中的内容，其余的承接上面的文件，里面的函数和类都不变

widget.cpp

这里只演示了不带参量的形式，带参量的需要对应更改函数指针，和之前相同。

这里一种写法connect(btn,&QPushButton::clicked,this,&Widget::classIsOver);这种写法相当于把点击操作连接到classIsOver这个函数上，在触发这个函数后，接着触发后面的过程

connect(btn,&QPushButton::clicked,zt,teachersignal2);这种写法是把信号连接到另一个信号上。

这是断开连接的操作

拓展
信号可以连接信号
一个信号可以连接多个槽函数
多个信号，可以连接同一个槽函数
信号和槽函数的参数必须类型一一对应
信号的参数个数可以多于槽函数的参数个数，但是存在对应的参数类型要对应

QPushButton

最基本的按钮，类似按钮控件还有ToolButton。QPushButton主要用于显示文字的按钮，ToolButton主要用于带图片的按钮。
